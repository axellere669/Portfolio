SAE S1.04

Création d'une base de données

A partir de l'histoire du Titanic, nous devions créer une base de données associée. L'objectif était de collecter des informations nous-mêmes, de réfléchir à une bonne organisation pour la base de données avec des tables pertinentes, et de voir si elle était utilisable. Nous étions en duo et avions plusieurs semaines pour réaliser cette SAE. PLusieurs documents étaient à rendre, notamment un rapport, la base de données, et un fichier réponse des requêtes SQL qui étaient demandées, répondant à des questions très précises sur notre base de données.


Ce que j'ai retenu de cette SAE :

  • Comment l'on passe d'un recueil de données à une mise en forme informatique
  • L'importance d'une base de données efficace, bien organisée, utilisable dans des cas complexes
  • L'étendue de questions qui peuvent être posées à une base de données, même petite


Compétences acquises finalement :

  • Créer une base de données
  • Schématiser, concevoir et réfléchir à l'organisation d'une base de données
  • Trier des informations, les représenter sous une autre forme
  • Sélectionner les informations pertinentes, que ce soit en amont ou en aval d'une base de données (pour la créer ou l'interroger)
